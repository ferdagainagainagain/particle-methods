## HW1 — Schelling's Model of Segregation

Implementation of a 2D variant of Schelling's (1971) model on a 100×100 toroidal grid.

**Files:**
- [`hw1/hw1.ipynb`](hw1/hw1.ipynb) — simulation notebook (all code + plots)
- [`hw1/report.tex`](hw1/report.tex) — LaTeX report source
- [`hw1/homework1/hw1.pdf`](hw1/homework1/hw1.pdf) — compiled report
- [`hw1/figures/`](hw1/figures/) — exported figures (PDF)

**Topics covered:**
- Periodic boundary conditions (toroidal grid)
- Four update/movement strategy combinations
- Phase transition at H = 4 → 5

**Requirements:** Python 3, NumPy, Matplotlib
